# Build Project ZEON - Autonomous AI Operating System (Zero Budget)

You are a senior AI architect and engineer.

Your task is to build an autonomous AI operating system inspired by ZEON and FRIDAY.

This is NOT a chatbot.

This is NOT an LLM wrapper.

This is a persistent autonomous intelligence platform.

The objective is to create a self-improving AI capable of:

* Understanding goals
* Planning tasks
* Executing tasks
* Learning from outcomes
* Controlling a computer
* Browsing the internet
* Building software
* Researching topics
* Maintaining long-term memory
* Improving workflows over time

The system must be designed for a solo developer with zero budget.

No paid APIs.

No proprietary services.

Everything must run locally.

---

# Core Philosophy

LLMs are tools.

They are not the AI.

The AI consists of:

Reasoning

* Memory
* Planning
* Learning
* Perception
* Execution
* Reflection
* Knowledge
* Agency

The LLM is only one component.

Design the system accordingly.
